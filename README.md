# CSS Practice

Small collection of HTML/CSS examples and exercises used for learning and experimentation.

## Overview
- Purpose: practice layout, responsive design, and CSS features.
- Contents: HTML files, CSS styles, and assets.

## Live preview
- Open `index.html` in your browser.
- Or serve locally (recommended):
  - Using VS Code Live Server extension: Open folder → Open `index.html` → "Live Server".
  - Using Python (PowerShell/CMD):  
    `python -m http.server 8000`  
    then open http://localhost:8000

## Project structure
- index.html — main demo
- css/ — stylesheets
- assets/ — images/fonts (if present)
- README.md — this file
